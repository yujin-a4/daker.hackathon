# **App Name**: VibeHack

## Core Features:

- Hackathon Listing & Filtering: Browse active, upcoming, and ended hackathons with options to filter by status or tags.
- Detailed Hackathon View: View comprehensive information for each hackathon, including rules, schedule, prizes, and team policies.
- Team Formation & Camp: Discover open teams, join an existing team, or create a new team for a specific hackathon, managed locally via Zustand.
- Live & Final Leaderboards: Access the leaderboard for each hackathon, displaying team rankings, scores, and submission details.
- Project Submission Portal: Submit required artifacts (e.g., zip files, PDFs, URLs) for participating in hackathons, managed locally via Zustand.
- User Rankings Profile: View overall user rankings, displaying points, number of hackathons joined, and wins.
- Hackathon Idea Assistant: A generative AI tool to help participants brainstorm project ideas based on the hackathon's theme and constraints.

## Style Guidelines:

- A dark color scheme suggesting a modern, tech-forward, and energetic platform.
- Primary color: Deep technical blue (`#1F4966`). This hue embodies precision and depth, suitable for a coding and challenge-oriented platform.
- Background color: A very dark desaturated blue (`#16191B`), providing a stark yet subtle foundation for content, making lighter elements stand out.
- Accent color: Vibrant cyan-blue (`#3DC5E8`). This contrasting, energetic hue draws attention to interactive elements, calls to action, and important highlights.
- Headline and body text for Korean: 'Pretendard' (sans-serif), ensuring readability and a modern aesthetic.
- Headline and body text for English: 'Inter' (sans-serif), chosen for its legibility across various screen sizes and its clean, professional look.
- Note: currently only Google Fonts are supported.
- Utilize 'Lucide React' icons for a consistent, crisp, and modern iconography system throughout the application.
- A responsive layout ensuring optimal viewing and interaction across desktop and mobile devices, with clear navigation for hackathons, teams, and rankings.
- Implement subtle and engaging animations using 'Framer Motion' for transitions, interactive elements, and overall user experience enhancement, without being distracting.
